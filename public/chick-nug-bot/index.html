Introduction

The day is January 26, 2022. I have been coding for about a year and have became quite good. I decide to start experimenting external API's. I had done it before, making useless Discord bots. But I wanted to try something new. I decided to start experimenting with the Twitter API. Incase you don't know, an API is a way to get and send data to a service (such as Twitter) via code. I spend a while thinking about what I will want it to do, a bot. I think that I should try something simple. And then the idea was born. A Twitter bot that will make a tweet every 30 minutes containing "[number] pc chicken nugget." Every time it tweets, the number goes up. I then get to work.
